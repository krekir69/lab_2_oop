2. Шаблонные классы
1. Не пользуясь шаблонами библиотеки STL, реализовать класс-шаблон TVector (вектор). В данном  классе необходимо реализовать:
•	конструктор с параметром «размерность» (по умолчанию = 1)
•	конструктор копий
•	деструктор
•	метод setLength, позволяющий изменять длину вектора
•	оператор присваивания
•	оператор += сложения двух векторов (одинаковой размерности)
•	оператор индексации [ ]
•	friend-оператор << вывода вектора в поток
•	friend-оператор >> ввода вектора из потока.
В функции main продемонстрировать все возможности TVector (в том числе, использование оператора [ ]) для типов double и. string (STL).
2. Не пользуясь шаблонами библиотеки STL реализовать шаблонный класс TMatrix (прямоугольная матрица).
Класс TMatrix должен быть контейнером объектов TVector на основе композиции и содержать динамический массив объектов TVector (одинаковой длины). В  классе TMatrix необходимо реализовать:
•	конструктор с параметрами «число строк» и «число столбцов» (по умолчанию = 1), который создает матрицу заданной размерности
•	конструктор копий
•	деструктор
•	оператор присваивания (для матриц одинаковой размерности)
•	оператор += сложения двух матриц (одинаковой размерности)
•	оператор индексации [ ]
•	friend-оператор << вывода матрицы в поток
•	friend-оператор >> ввода матрицы из потока.
В функции main продемонстрировать все возможности  TMatrix (в том числе, использование оператора [ ]) для типов int и CStr из 1-го задания.
